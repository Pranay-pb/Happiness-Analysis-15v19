# 🌍 Global Happiness Analysis (2015 vs. 2019)
A comparative data analysis project using Python to explore how global happiness

## 📌 Project Overview
This project analyzes World Happiness Report data from **2019** and **2015**. By leveraging Python's data science stack, I investigate how variables like GDP, social support, and life expectancy correlate with overall happiness scores across different eras.

## 📊 Key Features
*   **Data Cleaning:** Handling missing values and aligning schemas between 2019 and 2025 datasets.
*   **Exploratory Data Analysis (EDA):** Statistical summaries of happiness trends.
*   **Comparative Visualisation:** Side-by-side plots showing shifts in global well-being.
*   **Correlation Mapping:** Identifying which factors (e.g., Freedom, Generosity) became more or less influential over time.

## 🛠️ Technologies Used
*   **Python 3.x**
*   **Pandas:** Data manipulation and analysis.
*   **NumPy:** Numerical computing.
*   **Matplotlib & Seaborn:** Static and heat-map visualisations.

## 📂 Dataset
The analysis is based on two CSV files:
*   `happiness_2019.csv`: Data reflecting pre-pandemic global happiness.
*   `happiness_2025.csv`: Latest data reflecting current global trends.

*Note: Ensure these files are placed in the `data/` folder before running the notebook.*

## 🚀 Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Install dependencies:**
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. **Run the analysis:**
   Open `analysis.ipynb` in Jupyter Notebook or VS Code and run all cells.

## 📈 Visualisations Included
*   **Heatmaps:** Showing correlation changes between 2019 and 2025.
*   **Bar Charts:** Top 10 happiest countries comparison.
*   **Scatter Plots:** Relationship between GDP per capita and Happiness Score.
